# Frontend

Frontend simple para la entrega.

## Paginas recomendadas
- `index.html`
- `login.html`
- `register.html`
- `events.html`
- `my-events.html`

## Tecnologia
- HTML
- CSS
- JavaScript vanilla
- `fetch` para consumir los microservicios

## Orden de trabajo
1. Maquetar las paginas.
2. Consumir `auth-service` para login y registro.
3. Consumir `event-service` para listar eventos.
4. Consumir `inscrip-service` para inscribirse y ver mis eventos.
